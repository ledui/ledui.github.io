Green Code

Green Code is an LED based textual representation that aims to be useful in LED applications.

Documentation:

  • Introduction to Green Code
  • Green Code Dictionary - all of the currently defined characters in standard (RGB) form of the language.
  • Green Code Hundred words - Here is a list of 100 common English words, many of which are short. I seem to find that long words can be easier to read since after decoding a few letters, you can employ your 'hangman' skills.
  • White Code - Alternate one colour format
